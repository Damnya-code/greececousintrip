# Travel Log media

Create one folder per day, for example `assets/log/day-03/`, and use ordered descriptive names such as `01-harbour.webp`, `02-breakfast.webp` and `03-evening-walk.avif`.

Keep originals outside the public repository. Add only web-sized copies here: normally 1600–2400 pixels on the long edge, with smaller responsive variants when an entry becomes image-heavy. Prefer WebP or AVIF, record the actual width and height in the day data, and never commit an image that must remain private.

See `docs/TRAVEL_LOG_GUIDE.md` for the editing workflow and copyable block examples.
